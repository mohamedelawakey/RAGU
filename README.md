# RAGU

**RAGU** is a simple and powerful project designed to make interacting with educational files and books easier. It uses **RAG** (Retrieval-Augmented Generation) technology to provide accurate answers based on your own content.

## What is it for?
The main goal of this project is to transform information within documents (such as PDFs and articles) into a "knowledge base" that you can chat with and query instantly.

## Use Cases
- **Studying:** Upload your textbooks or lectures and ask about specific topics; it will extract the answer directly from the content.
- **Research:** Quickly find specific information across multiple files without having to read through all of them.
- **Summarization:** Ask for summaries of specific sections or connect different pieces of information within your files.

## How to use it?

### 1. Clone the repository
Get the project to your local machine:
```bash
git clone https://github.com/mohamedelawakey/RAGU.git
cd "RAGU"
```

### 2. Automatic Setup (Cross-Platform)
Run our unified script to automatically create a virtual environment and install all necessary dependencies (Works on Linux, Windows, and macOS).

**On Linux/macOS:**
```bash
chmod +x run_all.sh
./run_all.sh
```

**On Windows:**
```bash
./run_all.sh
```

### 3. Start Processing
1. Place your documents (PDFs, etc.) in the data folder.
2. Run your pipeline scripts to start answering questions.
